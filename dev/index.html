Sesam.smoothstep — Function
smoothstep(x, x_step, dx, a=zero(x), b=one(x))

smooth step function:

  • = a for x <= x_step - dx
  • = b for x >= x_step + dx
  • = (a-b)/2 for x == x_step

and smooth in between.
